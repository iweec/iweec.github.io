官网

-[burn.hair](https://burn.hair/)

介绍（取自官网）

  • OpenAI 接口转发站

  • 正规渠道，稳定使用，所有模型均支持高并发，适合沉浸式翻译

  • 支持 OpenAI 大部分模型，可以看这里，包括 gpt-4-turbo-2024-04-09 和 gpt-4o-2024-05-13

使用指南

令牌创建

  • GitHub 登录： 确保您已拥有一个 GitHub 账号，没有的话去Github注册

  • 创建令牌： 登录后，在设置区域寻找到 令牌 ，并选择 添加新的令牌

  • 复制令牌： 令牌创建后复制，以便在 API 请求中使用，当然，你也可以直接点击 聊天

API调用

Open AI 接口

获取支持模型列表:

  • GET https://burn.hair/v1/models

  • 请求头 Authorization: Bearer sk-xxxx

DALL-E

  • POST https://burn.hair/v1/images/generations

  • 请求头 Authorization: Bearer sk-xxxx

  • 请求体为json

获得额度

  • 签到：通过网站或机器人进行签到，机器人签到可能会获得额外奖励

  • 充值: 访问 充值 购买兑换码。 购买后，使用获得的兑换码在充值页面进行激活

  • 首次关注telegram频道可以领取免费兑换码
